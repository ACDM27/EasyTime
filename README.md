# EasyTimer - 轻量级在线时间管理工具

[Project Banner](https://via.placeholder.com/800x200?text=EasyTimer+Demo)

## 项目概述
一个基于web的轻量级计时工具，提供精准计时和倒计时功能，支持响应式设计，适用于学习、工作和运动等多种场景。

## 核心功能

## 基础计时
- 启动/暂停计时控制
- 一键重置功能
- 时间显示格式：`HH:MM:SS`

## 倒计时模式
- 自定义时长设置（支持小时/分钟/秒）
- 结束提醒（声音提示 + 弹窗提示）
- 进度条可视化

## 用户界面
- 响应式布局
- 简洁设计
- 主题颜色可改变

## 技术栈
- 前端框架Vue3
- UI优先使用原生CSS，部分引入组件库Element Plus
- 状态管理使用Pinia

## 安装依赖
npm install 

## 启动运行
npm run serve

## 生产构建
